# 🚀 Space Radar

ディスク使用量を視覚的に分析する Windows デスクトップアプリケーションです。  
フォルダーをスキャンし、パイチャートとリストでファイル・フォルダーのサイズを一目で把握できます。

## ✨ 機能

### 🔍 フォルダースキャン
- Windows API（`FindFirstFileW`）による高速スキャン
- スキャン中はステータスバーに現在処理中のパスをリアルタイム表示
- キャンセル対応の非同期処理

### 📊 パイチャート表示
- スキャン結果をパイチャートで可視化（[ScottPlot](https://scottplot.net/) 使用）
- 各スライスに色が割り当てられ、フォルダーリストと連動

### 📁 フォルダーリスト
- 現在のフォルダー配下のファイル・サブフォルダーを一覧表示
- アイコンでファイル／フォルダーを識別
- ダブルクリックでサブフォルダーにドリルダウン
- 右クリックメニューからエクスプローラーで開く

### 🧭 ナビゲーション
- ツールバーの **「⬆ 上へ」** ボタンで親フォルダーへ戻る
- パンくずリストで現在のパスを表示
- 合計サイズをリアルタイム表示

### 📋 Top N 大きいファイル
- **「📊 Top N」** ボタンでパネルを開閉
- 表示件数を 10 / 20 / 50 件から選択
- **「📁 現在のフォルダーのみ」** トグルでスコープを切り替え（全体 or 現在のフォルダー以下）
- ランク・ファイル名・パス・サイズバー・サイズを一覧表示
- 右クリックメニューからエクスプローラーで開く

### 🖱️ ドラッグ＆ドロップ
- ウィンドウにフォルダーをドロップするだけでスキャン開始
- ドロップ中はオーバーレイ表示

## 🚀 始め方

### 必要環境

- Windows 10 / 11
- [.NET 10 SDK](https://dotnet.microsoft.com/download)

### ビルドと実行

```bash
git clone https://github.com/uta666666/SpaceRadar.git
cd SpaceRadar
dotnet run --project SpaceRadar
```

---

## 📖 使い方

1. **フォルダーを選択**: ツールバーの「📂 フォルダーを選択」をクリック、またはフォルダーをウィンドウにドロップ
2. **スキャン完了後**: パイチャートとフォルダーリストにサイズが表示されます
3. **ドリルダウン**: リストのフォルダーをダブルクリックしてサブフォルダーを確認
4. **上に戻る**: 「⬆ 上へ」ボタンで親フォルダーへ移動
5. **Top N 確認**: 「📊 Top N」ボタンで大きいファイルの一覧を表示
