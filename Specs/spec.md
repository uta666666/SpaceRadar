# Space Radar

## Introduction

Space Raderは、Windowsのフォルダーをレーダーのように表示するソフトウェアです。フォルダーのサイズや使用状況を視覚的に把握することができます。

## Features
- フォルダーのサイズを円グラフで表示
- フォルダーを選択することで、そのフォルダー内の情報に表示を切り替えることができる
- フォルダーのサイズを色分けして表示

## Usage
1. Space Raderを起動します。
1. 表示したいフォルダーを選択します。
1. 選択したフォルダーのサイズや使用状況がレーダー形式で表示されます。
1. フォルダーをクリックすることで、そのフォルダー内の情報に表示を切り替えることができます。

## Tecnologies Used
- .NET10 - アプリケーションのフレームワークとして使用
- C# - ソフトウェアの開発に使用
- WPF - ユーザーインターフェースの構築に使用
- ScottPlot - グラフの描画に使用
- ReactiveProperty - データバインディングに使用
- フォルダーの情報を取得するためのWindows API - フォルダーのサイズや使用状況を取得するために使用（なるべくWindows APIを使用して、パフォーマンスを向上させることを目指しています）

## Airchitecture
Space Raderは、Model-View-ViewModel (MVVM) アーキテクチャを採用しています。これにより、コードの分離と再利用性が向上し、保守性が高まります。
- Model: フォルダーの情報を管理するクラス
- View: ユーザーインターフェースを構築するXAMLファイル
- ViewModel: ModelとViewを仲介するクラスで、データバインディングを使用してViewにデータを提供します。
- Services: フォルダーの情報を取得するためのサービスクラス
- Utilities: 共通のユーティリティクラス
- Commands: ユーザーの操作に対応するコマンドクラス
- Converters: データの変換を行うクラス
- Resources: アプリケーションのリソースを管理するフォルダー
